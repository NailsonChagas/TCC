# Gerenciador da plataforma HIL

## Preparação e Execução

Abra o terminal na pasta do projeto e execute os comandos abaixo para configurar o ambiente:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```

Com o ambiente ativado e as dependências instaladas, inicie o programa:

```bash
python main.py

```

## Notas

* Para encerrar o ambiente virtual, utilize o comando `deactivate`.
* Certifique-se de que o ficheiro `data.json` permanece na pasta para manter os seus dados salvos.
