# Plataforma HIL para Validação de Controladores Digitais em Conversores Buck

Este repositório contém os arquivos relacionados ao desenvolvimento do **Trabalho de Conclusão de Curso (TCC)** de **Nailson Francisco da Silva Chagas**, cujo objetivo é o desenvolvimento de uma plataforma **Hardware-in-the-Loop (HIL)** para validação de controladores digitais embarcados aplicados a conversores **DC-DC do tipo buck**.
